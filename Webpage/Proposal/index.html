Problem Description

The path-tracing algorithm we implemented in class tries to render photo-realistic images. We want to render images in an artistic style, so we will implement non-photorealistic rendering and other shader types. We will have to modify our path-tracing algorithm. It will be challenging to get our rendering to look artistic.


Goals and Deliverables

Part 1 (what we must accomplish): Successfully implement NPR and render an artistic image. Our baseline will be at least one shader per person, to be applied to a single object and rendered into a single image.

Part 2 (what we hope to accomplish): Our final deliverable will showcase images of a 3D scene or model applied with different types of shaders and non-photorealistic renders. We hope to be able to apply the shader to more complex scenes with multiple objects, backgrounds, etc. and possibly add more shaders.
